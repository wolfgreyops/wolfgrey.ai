Workflow Types

See how agents work together

Click a workflow type to see how agents coordinate to complete complex tasks.

Linear Workflow

Tasks flow sequentially from one agent to the next. Each agent completes its work before passing to the next stage.

Best for: Lead nurturing, content pipelines, onboarding sequences

Parallel Processing

Multiple agents work simultaneously on different aspects of a task, then results are combined.

Best for: Content creation, proposal generation, competitive analysis

Hierarchical Workflow

Manager agents delegate to specialist agents, who may delegate further. Results roll up through the hierarchy.

Best for: Complex operations, multi-department coordination, enterprise workflows

Event-Driven Workflow

Agents react to events in real-time. When something happens, the appropriate agent is triggered automatically.

Best for: Customer support, notifications, real-time integrations
